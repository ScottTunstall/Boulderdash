Decoded cave data in human-readable text, including ascii representation. Warning: There may be mistakes in the decode. The C source code to generate the following decodes are available in the files decodecaves.c and cavedata.h). Note that when decoding the cave, the random objects are placed first, then a bounding rectangle of steel wall is put in (effectively an implicit DrawRect(0,2,40,22) before the cave data), then the cave data is decoded. For each cave, a graphical representation is given for difficulty level 1. The characters used in the graphical representation are the ones used by a implementation of BoulderDash on the Amiga.
